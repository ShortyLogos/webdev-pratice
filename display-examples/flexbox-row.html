FLEX-ROW

Setting a container is required when using flexbox. First, set its display to flex. The flex-direction by default is row. The span tag is used with inline elements whilst the div tag is used with block-level content. When trying to center an element, margin: auto is always a good try.

flex-direction: row (default)

flex-wrap: wrap
