BLOCK

HTML elements are block or inline elements by default. Block elements put themselves on a new line. Text tags (h1, p, etc.) are block. Pictures and links are inline. The span tag is used with inline elements whilst the div tag is used with block-level content. For centering an element, margin: auto is always a good try.

text-align: center

Pictures are inline by default.
